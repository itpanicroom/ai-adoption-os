# Skill: Champions Program

> **Status:** Placeholder — not implemented in v0.1

## Purpose

The Champions Program skill designs and activates a network of internal AI advocates — employees who use the technology enthusiastically, help their colleagues, and create sustainable peer-to-peer adoption momentum.

Champions are the most cost-effective adoption accelerator available to organizations deploying AI.

## CAF Phase

**Plan · Adopt · Manage**

## Invoked By

- Readiness Agent
- Adoption Agent

## Inputs

- Organizational input profile (champions program status, departments, user count)
- Stakeholder map (from Stakeholder Mapping skill)
- Scenario Pack: Role-based use cases to equip champions with

## Expected Outputs

- Champions program design document
- Champion selection criteria and recruitment plan
- Champion onboarding plan
- Champion activation toolkit (talking points, use case examples, FAQ)
- Champion engagement cadence (meetings, recognition, feedback loops)

## Key Activities

1. Define the champion profile (what makes a good champion in this organization)
2. Identify champion candidates from the stakeholder map and departmental contacts
3. Design the champion onboarding program
4. Equip champions with role-specific use case scenarios from the Scenario Pack
5. Define the ongoing engagement cadence for champions
6. Design recognition and incentive mechanisms to sustain champion engagement

## Champions Program Structure

| Component | Description |
|---|---|
| Selection | 1 champion per 20–25 users; cross-departmental coverage |
| Onboarding | Initial training on technology + adoption best practices |
| Toolkit | Use case library, FAQ, talking points, escalation path |
| Engagement | Monthly champion community calls; direct access to IT lead |
| Recognition | Internal visibility, early access to new features, executive acknowledgement |

## Notes

Organizations without a champions program at the start of v0.1 (as indicated by `championsProgram: false` in the input profile) should treat champion recruitment as a priority activity in the Plan phase. The absence of a champions program is one of the most reliable predictors of slow adoption.
